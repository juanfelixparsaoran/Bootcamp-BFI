SESI 5 : Object Oriented Programming

1. Program Bangun Datar
    Prinsip Inheritance dimana Persegi, Persegi Panjang, Lingkaran, dan Segitiga memiliki kesamaan method yaitu luas dan keliling
    dan mengextends kelas Bangun Datar.

2. Program Restoran
    Prinsip Enkapsulation dimana data suatu kelas tidak dapat diakses secara langsung dari luar,
    tetapi dengan method getter dan setter dari kelas tersebut.

3. Program Manusia
    Prinsip Polymorphism dimana suatu kelas bisa didefinisikan dari kelas lain yang berhubungan. ada 3
    dalam java.
    a. Class itu sendiri
    b. Class lain
    c. Interface

    Lalu ada konsep overriding dimana kemampuan method sama tetapi dengan perilaku yang berbeda dan
    konsep overloading dimana kemampuan method sama tetapi parameternya berbeda
