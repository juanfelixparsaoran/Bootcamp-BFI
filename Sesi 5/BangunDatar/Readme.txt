Petunjuk penggunaan program

1. Bangun Datar

Program digunakan untuk menghitung luas bangun datar berupa lingkaran, persegi, persegi panjang, dan segitiga
menggunakan konsep inheritance.

2. Manusia

Program ini digunakan untuk menampilkan perilaku manusia, baik cowok maupun cewek, dengan penggunaan konsep
polimorphism dan Overriding.

3. Restoran

Program ini digunakan untuk menampilkan detail menu dari restoran dengan menggunakan konsep encapsulation.

4. Overloading

Program ini digunakan untuk menampilkan nilai maksimum dari 2 angka, dengan konsep overloading untuk tipe
data int dan double.
